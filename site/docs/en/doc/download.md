### Use jad.sh

Download and unzip, find `jad.sh` in the directory. Start it in bash:

```bash
./jad.sh
```

Print usage:

```bash
./jad.sh -h
```

### Use jad-boot.jar

Download and unzip, find `jad-boot.jar` in the directory. Start with `java` command:

```bash
java -jar jad-boot.jar
```

Print usage:

```bash
java -jar jad-boot.jar -h
```
